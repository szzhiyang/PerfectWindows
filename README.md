# 想逃避无聊时光？想挣脱思维窠臼？想甩开现实包袱？

> * 这会启动默认浏览器。

# 想访问文件、控制面板、回收站、库、用户文件夹或网络设置？
# 按下【Home】键，回到文件的【家】！
> * 这会启动文件资源管理器窗口。        
> * 当然，你也可以将“此电脑”或者文件资源管理器固定至开始菜单。  
> * 想找回桌面文件，请进入用户文件夹中的 “桌面” 文件夹。          
> * 如果你使用某些笔记本电脑，你可能需要使用 Fn 键。  
> * 另外，你也可以使用 Win + E 组合键。

# 想快速打开你常用的文件夹、常用的应用程序或常用的设置选项？
# 按下【Win】徽标键，【赢】得高效！
> * 如果你正在使用 Windows 10 或 Windows 8.1，请右击你常用的项目，然后选择“固定至开始屏幕”。  
> * 如果你正在使用 Windows 7，请将你常用的项目拖动至开始按钮上。        
> * 由于开始菜单包容万象，随叫随到，且不会对你当前的工作或娱乐造成干扰，请尽量习惯去使用开始菜单。

# 想更快打开任何应用程序或任何设置选项？
# 按下【Win】徽标键后直接键入目标项目，【赢】下目标！

# 想锁定电脑？想让电脑小憩片刻？想让电脑酣睡一顿？
# 只需按下电源按钮或合上盖子！
> * 这会使电脑进入耗电量极低的睡眠状态。  
> * 睡眠状态持续 3 个小时后，电脑将自动进入完全不耗电的休眠状态。    
> * 如果你使用台式电脑并且不希望它持续耗电，切断其电源即可。    
> * 想继续使用电脑？按下电源按钮。

# 仍然适应不了？
# 没关系！
> 1. 找到批处理文件  
> 2. 右击批处理文件，选择 “重命名”
> 3. 在花括号中按照提示键入指定字母：
> ```
> 键入字母 u 以恢复桌面图标、任务栏图标、锁屏等界面元素，并取消浏览器和音量调整的快捷键
> 键入字母 m 以取消鼠标滚轮反向功能
> 键入字母 c 以重新启用 Cortana
> 键入字母 o 以重新启用 OneDrive
> 键入字母 n 以重新启用推送通知与通知中心
> 键入字母 p 以解锁电源选项
> ```
> 4. 按下回车键
> 5. 保存当前所有工作，因为稍后我们将重启电脑
> 6. 右击已被你重命名的批处理文件，选择 “以管理员身份运行”
